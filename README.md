👥 Employee Management System
A Java-based desktop application to manage employee records — built with Java Swing, MySQL & JDBC.

🔧 Tech Stack

Frontend: Java Swing (AWT/JFC)
Backend: Java
Database: MySQL + JDBC
IDE: Eclipse / IntelliJ / NetBeans


✨ Features

Secure Admin Login
Add, View, Update & Remove Employees
Search & Print Employee Records
Centralized MySQL Database


💻 Requirements

JDK 8+, MySQL Server, NetBeans/Eclipse
RAM: 4GB+, Processor: Intel i3 2.8GHz+


🚀 Setup

Clone the repo
Import database.sql into MySQL
Update DB credentials in conn.java
Add mysql-connector-java.jar to build path
Run Home.java
